# Code of Conduct

## Purpose

This repository is part of a theory-and-publication branch of the Terra
ecosystem.

The code of conduct exists to protect:

- clarity of meaning;
- responsible collaboration;
- respectful dialogue;
- publication integrity;
- child-safe and ethically bounded use of the material.

## Expected behavior

Contributors are expected to:

- work respectfully and without harassment;
- repair meaning and structure before multiplying files;
- discuss disagreements through evidence and trace, not noise;
- keep public-facing language readable and responsible;
- respect multilingual contributors while preserving an English-first public layer when appropriate.

## Unacceptable behavior

The following is not acceptable:

- harassment, insult, humiliation, or intimidation;
- deliberate vandalism of repository history or publication metadata;
- adding broken, deceptive, or placeholder service files and presenting them as complete;
- public disclosure of sensitive security details;
- exploitative or harmful use of child-facing or education-facing material.

## Repository-specific collaboration rule

This repository should not be used to dump uncontrolled artifacts.

If a file is damaged, duplicated, or out of place, the first duty is detox and
clarification, not further proliferation.

## Reporting

For conduct problems, use the maintainer contact routes already published in the
repository and Terra ecosystem surfaces.
